## Local Archive Boundary

Raw council drafts, obsolete composite plans, and other redundant working
material may be preserved under the ignored repository-root
`.local-archive/`. That directory is intentionally absent from clean clones,
must never be a canonical source or evidence dependency, and must not be linked
from active documentation.
